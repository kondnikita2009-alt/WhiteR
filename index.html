Задание 1

Очистка списка от четных чисел

Генерация случайного списка, создание его копии и фильтрация всех четных элементов с помощью генератора списка.

import random

n = int(input("Введите количество чисел: "))
min_val = int(input("Введите минимальное значение: "))
max_val = int(input("Введите максимальное значение: "))

original_list = [random.randint(min_val, max_val) for _ in range(n)]
print(f"\nИсходный список: {original_list}")

copied_list = original_list.copy()
copied_list = [x for x in copied_list if x % 2 != 0]

print(f"Только нечетные: {copied_list}")
Задание 2

Удаление по позициям

Удаление элементов на строго определенных позициях (2-й, 4-й, 6-й) с защитой от выхода за границы списка.

import random

n = int(input("Введите количество (мин. 6): "))
min_val = int(input("Мин: "))
max_val = int(input("Макс: "))

numbers = [random.randint(min_val, max_val) for _ in range(n)]
print(f"Исходный: {numbers}")

positions = [(1, "второй"), (3, "четвертый"), (5, "шестой")]

for index, name in sorted(positions, reverse=True):
    if index < len(numbers):
        removed = numbers.pop(index)
        print(f"Удален {name} ({removed})")

print(f"Итог: {numbers}")
Задание 3

Манипуляции со словами

Работа с методами insert() и append() для добавления элементов в начало, середину и конец списка.

n = int(input("Количество слов: "))
words = [input(f"Слово {i+1}: ") for i in range(n)]

words.insert(1, "апельсин") # На 2-ю позицию
words.append("лимон")      # В конец
words.insert(0, "слива")    # В начало

print(f"Новый список: {words}")
Задание 4

Удаление всех вхождений

Пользователь вводит список и значения, которые нужно из него полностью вычистить. Проверка на пустоту в конце.

nums = list(map(int, input("Числа через пробел: ").split()))
to_remove = list(map(int, input("Что удалить: ").split()))

for val in to_remove:
    while val in nums:
        nums.remove(val)

print("Список пуст!" if not nums else f"Итог: {nums}")

Свап и вставка в центр
